---
name: new-component
description: Scaffold a new React component following the project's components/ convention
---

Create a new component in components/<Name>/ with:
1. <Name>.tsx — the component
2. <Name>.module.css — scoped styles (empty, ready to use)
3. index.ts — barrel export: `export { default } from './<Name>'`

Follow the existing Navbar component as the reference pattern.
